dl : 설명과 관련된 목록 태그(주로 용어에 대해 정의할때)

HTML5
HTML5는 웹에서 정보를 표현할 목적으로 만든 마크업 언어다.
마크업 언어란 태그를 사용해 문서의 구조를 정의하는 언어다.
CSS3
HTML5와 함께 웹 표준의 하나다.
미리 약속된 스타일 속성을 사용해서 레이아웃과 스타일을 정의한다.
